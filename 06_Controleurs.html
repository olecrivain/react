Contrôleurs

https://docs.angularjs.org/guide/controller

Les contrôleurs - 1

La méthode .controller() permet d'enrichir un module Angular avec un nouveau contrôleur.

Quand un nouveau contrôleur est instancié, un nouvel objet $scope est créé puis injecté dans le contrôleur.


var cra = angular.module('cra', []);
cra.controller('Page1Controller', ['$scope', function($scope) {
}]);

Les contrôleurs - 2

Ils permettent d'initialiser l'état de l'objet $scope


var cra = angular.module('cra', []);
cra.controller('Page1Controller', ['$scope', function($scope) {
  $scope.dates = ['2016/03/10', '2016/03/11', '2016/03/12'];
  $scope.now = new Date();
}]);

Les contrôleurs - 3

Les propriétés et méthodes de l'objet $scope sont accessibles depuis le template :


<div ng-controller="Page1Controller">
  {{dates}}
  {{now}}
</div>

Les contrôleurs - 4

Ils permettent aussi de définir du comportement


var myApp = angular.module('myApp',[]);

myApp.controller('DoubleController', ['$scope', function($scope) {
  $scope.double = function(value) { return value * 2; };
}]);
          

<div ng-controller="DoubleController">
  Two times <input ng-model="num"> equals {{ double(num) }}
</div>
