Data binding

Angular synchronise automatiquement le modèle (JavaScript) avec la vue (DOM), et inversement.

Angular automatise les modifications dans le DOM quand le modèle est modifié.

Angular automatise les modifications dans le modèle quand la vue est modifiée.

Data binding - Exemple


<div ng-app ng-init="qty=1;cost=2">
  <b>Invoice:</b>
  <div>
    Quantity: <input type="number" min="0" ng-model="qty">
  </div>
  <div>
    Costs: <input type="number" min="0" ng-model="cost">
  </div>
  <div>
    <b>Total:</b> {{qty * cost}}
  </div>
</div>
