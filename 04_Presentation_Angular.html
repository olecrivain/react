Présentation du framework

Historique de l'écosystème JavaScript

  • Avant 2005 : Pas de framework, difficile de gérer les différences entre navigateurs, pas d'outillage.
  • 2005 : jQuery. Efface les différences entre navigateurs. Manipulation du DOM simplifiée.
  • 2010 : Angular. Simplifie grandement le dev d'application type SPA.
  • 2013 : React. Orienté composant.
  • 2016? : Angular 2. Très différent de la v1. Orienté composant, Typescript...

Structure type d'un projet Angular - 1

Classé par type.


├── app
│   ├── controller
│   │   ├── page_1.js
│   ├── views
│   │   ├── page_1.html
│   ├── css
│   │   ├── styles.css
│   ├── app.js
├── .eslintrc.js
├── bower.json
├── package.json
├── index.html

Structure type d'un projet Angular - 2

Orienté composant. C'est la structure recommendée.


├── app
│   ├── components
│   │   ├── page_1
│   │   │   ├── page_1.js
│   │   │   ├── page_1.html
│   │   │   ├── page_1.css
│   ├── services
│   ├── app.js
├── .eslintrc.js
├── bower.json
├── package.json
├── index.html

app/app.js

Initialisation du module Angular de l'application "cra".


angular.module('cra', []);

Le contrôleur app/page_1/page_1.js


angular.module('cra').controller('Page1Controller', ['$scope', function($scope) {

  $scope.name = 'Olivier';

}]);

Le template index.html


<!doctype html>
<html ng-app="cra">
  <head>
    <script src="bower_components/angular/angular.min.js"></script>
    <script src="app/app.js"></script>
    <script src="app/page_1/page_1.js"></script>
  </head>
  <body>

    </div>
  </body>
