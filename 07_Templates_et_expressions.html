Templates

Les templates Angular sont des fichiers HTML, enrichis avec :

  • Les directives : balises et attributs Angular (ng-model,...)
  • Les expressions : {{...}}

Directives - Exemples 1


<input type="text" ng-model="name" ng-value="Elsa"/>

<div ng-show="true">visible</div>

<div ng-hide="1 === 1">invisible</div>

<div ng-class="var1 + '-color'">abc</div>

<button ng-click="alert('test');" ng-disabled="true">click!</button>

Directives - Exemples 2


<span ng-repeat="n in [1, 2, 3]">
  {{n}}
</span>

Directives - 3

La documentation de référence avec toutes les directives Angular : https://docs.angularjs.org/api/ng/directive

Les expressions

https://docs.angularjs.org/guide/expression

Les expressions - 1

Syntaxe {{...}} dans le html : 


<span title="{{expression}}">
  {{expression}}
</span>

Sans accolades dans les attributs des directives : 


<button ng-click="maFonction()">
  Ok
</button>

Les expressions - 2

Les expressions sont évaluées comme du code JavaScript, en utilisant le $scope comme contexte. Exemples :

  • 1+2
  • a+b
  • user.name
  • items[2].id

Les expressions - 3

Différences entre les expressions Angular et JavaScript :

  • Appeler une propriété sur un objet null ou undefined ne lève pas d'exception. Retourne une chaîne vide.
  • Les boucles et les conditions ne sont pas autorisées (mais l'opérateur ternaire est autorisé). L'idée est d'éviter de mettre trop de logique dans les templates

Les expressions - 4

Différences entre les expressions Angular et JavaScript :

  • Il n'est pas possible de créer des objets avec new.
  • Il est possible d'utiliser les filtres Angular.
  • Pas d'accès aux variables globales windows, document, location.
