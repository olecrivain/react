Environnement de développement

Node

Node est un interpréteur JavaScript (https://nodejs.org). Il est utilisé pour exécuter du code JavaScript en dehors d'un navigateur. 


$ node script.js

NPM (Node Package Manager)

Fourni avec Node. Il permet de :

  • Installer des outils exécutés par Node (ex : Grunt, eslint...)
  • Installer les dépendences d'un projet Node
  • Exécuter des scripts dans un projet Node

  • Code : https://code.visualstudio.com
  • Sublime Text : https://www.sublimetext.com
  • Atom.io : https://atom.io

Linter JS

Ils permettent :

  • d'identifier des patterns de code dangereux
  • d'eviter certaines typos
  • de rendre le code plus consistent (dans le cas d'eslint)

Linter JS - Plusieurs solutions

  • ESLint : le plus récent (celui qu'il faut utiliser)
  • JSLint
  • JSHint : le premier

ESLint

Installation globale : 


$ npm install -g eslint
$ eslint plus_ou_moins.js

Installation locale : 


$ npm install eslint --save
$ npm run lint

Le fichier .eslintrc.js à la racine du projet permet de configurer ESLint.

Exemple de fichier .eslintrc.js


module.exports = {
  "env": {
    "browser": true,
    "node": true
  },
  "extends": "eslint:recommended",
  "rules": {
    "indent": [ "error", 2 ],
    "quotes": [ "error", "single" ],
    "semi": [ "error", "always" ],
    "no-console": 0
  }
};

Liste des règles disponibles : http://eslint.org/docs/rules/.

ESLint & Atom.io

Les retours d'ESLint peuvent être directement intégrés dans Atom.io. Il faut pour cela installer les plugins : linter et linter-eslint.
